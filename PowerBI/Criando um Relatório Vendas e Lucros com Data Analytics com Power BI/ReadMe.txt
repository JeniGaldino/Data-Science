Relatório de Vendas e Lucros
Este relatório foi desenvolvido no Power BI para analisar o desempenho de vendas e lucros de uma empresa. Ele utiliza técnicas avançadas de agrupamento de dados, segmentação, formatação condicional e cálculos personalizados para fornecer insights claros e acionáveis.

Objetivo do Relatório
O objetivo principal deste relatório é:

Analisar o desempenho de vendas e lucros por segmento, país e outros critérios.

Identificar os principais contribuintes para as vendas e lucros.

Fornecer uma visão clara e segmentada dos dados para facilitar a tomada de decisões.

Funcionalidades Principais
1. Agrupamento de Segmentos
Os segmentos foram organizados em dois grupos principais: "Destaque" (Government e Small Business) e "Outros" (Channel Partners, Enterprise e Midmarket).

Essa segmentação permite uma análise focada nos segmentos mais relevantes.

2. Identificação dos Principais Países
O relatório identifica os Top 3 países com as maiores vendas, destacando os mercados mais importantes.

3. Análise de Vendas e Lucros
Foram criadas métricas personalizadas para calcular o total de vendas, lucro e média de vendas por segmento.

Essas métricas permitem uma análise detalhada do desempenho financeiro.

4. Visualização Segmentada
As barras nos gráficos de vendas são coloridas de acordo com o grupo ao qual pertencem (Destaque ou Outros), facilitando a identificação visual dos segmentos mais relevantes.

5. Formatação Condicional
Destaques visuais são aplicados para chamar a atenção para informações importantes, como países com maiores vendas ou segmentos com maior lucratividade.

6. Organização dos Dados
Os dados são organizados em diferentes páginas e visualizações, permitindo uma análise detalhada por segmento, país e outras dimensões.

Tecnologias e Técnicas Utilizadas
Agrupamento de Dados: Para segmentar os dados em grupos relevantes.

Cálculos Personalizados: Para criar métricas como total de vendas, lucro e média de vendas.

Formatação Condicional: Para destacar informações importantes nos gráficos e tabelas.

Hierarquias: Para organizar os dados em níveis, como segmento principal e subsegmento.

Como Utilizar o Relatório
Filtros Interativos:

Use os filtros na lateral para selecionar países, segmentos ou períodos específicos.

Navegação entre Páginas:

Navegue entre as páginas do relatório para visualizar diferentes perspectivas dos dados.

Clique nos gráficos para destacar informações específicas e explorar os dados em detalhes.
